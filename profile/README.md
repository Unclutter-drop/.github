# Unclutter Software — Portable Drop-Down Panel

## About Unclutter Software

**Unclutter Software** provides a complete drop-down panel from the top of your screen for notes, clipboard history, and file storage — for users tired of switching between applications. **Unclutter Software** combines quick notes, clipboard manager, and temporary file shelf into one cohesive interface. **Unclutter Software** supports text snippets, clipboard history, and drag-and-drop file storage. **Unclutter Software** includes iCloud sync, dark mode, and keyboard shortcuts. **Unclutter Software** enables users to access three essential tools without leaving their current app. **Unclutter Software** is trusted by users who want unified access to notes, clips, and files.

## Deep Dive — Unclutter Software Notes

Drop-down panel — swipe from top edge or press shortcut, panel slides down
Notes section — write quick notes, markdown support, auto-save
Clipboard manager — history of copied items, search, re-copy
File storage — drag files to shelf, temporary or permanent storage
Keyboard shortcut — Cmd+Shift+U to show/hide panel
iCloud sync — sync notes and clips across Macs

## Quick Reference for Unclutter Software

| Operation | Unclutter Software Action |
|-----------|---------------------------|
| Show panel | Move cursor to top edge, or press Cmd+Shift+U |
| Hide panel | Click outside, move cursor away, or press Escape |
| Add note | Click notes section, start typing |
| Access clipboard | Click clipboard section, click any item to re-copy |
| Add file | Drag file to file storage section |
| Remove file | Drag file out or right-click > Remove |
| Search clipboard | Click search icon in clipboard section |
| Clear clipboard history | Right-click clipboard section > Clear |
